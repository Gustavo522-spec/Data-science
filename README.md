📌 Sobre o Projeto

Neste repositório, explorei a análise de um banco de dados de filmes. O objetivo principal aqui não foi construir um modelo final linear, mas sim me familiarizar com o ecossistema de análise de dados em Python.

Aproveitei o material para testar diversas funcionalidades das bibliotecas, entendendo como manipular dados brutos e transformá-los em informações visuais.

🛠️ Ferramentas e Bibliotecas Utilizadas

Durante os exercícios e experimentos, utilizei:

* Pandas: Para leitura, filtragem e tratamento dos dados coletados.

* NumPy: Essencial para cálculos estatísticos, como a busca por médias e medianas.

* Seaborn & Matplotlib: Utilizados para a visualização dos dados, criando gráficos que facilitam a compreensão dos insights.

* Google Colab: Ambiente de desenvolvimento para execução dos notebooks.

🚀 O que eu pratiquei

* Importação e manipulação de datasets;

* Limpeza e tratamento de variáveis;

* Exploração de diferentes tipos de visualizações gráficas;

* Cálculos de estatística básica para análise exploratória.


⚠️ AVISO

este é um repositório de aprendizado contínuo, onde priorizei a experimentação de comandos e a prática direta com as bibliotecas
